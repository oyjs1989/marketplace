# 决策支持集成技能 (Decision Support Integration Skill)

**版本**: 1.0.0
**语言**: 中文 (Chinese)

综合性决策支持技能,提供多准则决策分析、方案比较和综合推荐。可作为独立技能使用,也可作为其他技能的集成器。

## 快速开始

### 触发关键词

- "帮我决策"
- "比较方案"
- "选哪个方法"
- "决策矩阵"
- "评估备选方案"
- "权衡分析"
- "技术选型"
- "方案对比"

### 示例使用

**场景1: API网关选型**
```
用户: "帮我选择API网关:Kong vs Nginx vs AWS API Gateway"

系统: 执行决策分析流程
1. 识别备选方案(3个)
2. 定义评估标准(性能、功能、成本、易用性等)
3. 确定权重
4. 对每个方案评分
5. 计算加权总分
6. 敏感度分析
7. 综合推荐

输出: 完整的决策分析报告(中文) + 推荐方案 + 实施建议
```

**场景2: 架构方案决策(集成模式)**
```
用户: "比较微服务架构 vs 模块化单体架构"

系统: 综合分析
1. 调用 problem-solving 技能分析两种架构
2. 调用 risk-assessment 评估各方案风险
3. 调用 cost-benefit-analysis 计算经济性
4. 整合所有分析到决策矩阵
5. 提供风险调整后的推荐

输出: 多维度决策报告 + 综合推荐
```

## 核心能力

### 1. 多准则决策分析 (MCDA)
- **加权评分法**: 最常用,直观易懂
- **AHP (层次分析法)**: 复杂层次化决策
- **TOPSIS**: 大量方案的客观排序
- **决策树**: 顺序决策和不确定性场景

### 2. 方案比较
- 定义评估标准和权重
- 量化方案表现
- 可视化比较结果
- 敏感度分析

### 3. 集成能力
- 整合 `risk-assessment` 风险评估
- 整合 `cost-benefit-analysis` 经济分析
- 整合 `problem-solving` 方案生成

## 决策流程

```
明确决策目标
  ↓
识别备选方案
  ↓
定义评估标准
  ↓
确定标准权重
  ↓
评分和分析
  ↓
[可选] 调用风险评估
  ↓
[可选] 调用成本效益分析
  ↓
敏感度分析
  ↓
综合推荐
```

## 输出格式

标准化的中文决策报告:

1. **决策目标** - 明确决策上下文
2. **备选方案** - 所有候选方案及优劣势
3. **评估标准** - 标准定义和权重理由
4. **评分矩阵** - 详细的评分和加权计算
5. **详细分析** - 每个方案的深入分析
6. **风险分析** - 集成的风险评估结果
7. **成本效益分析** - 集成的经济分析
8. **敏感度分析** - 决策稳健性评估
9. **推荐方案** - 首选方案及理由
10. **实施建议** - 可操作的下一步

## 决策方法选择

| 方法 | 适用场景 | 优势 | 局限 |
|------|----------|------|------|
| 加权评分法 | 3-7个方案,5-10个标准 | 直观易懂,快速 | 主观性较强 |
| AHP | 复杂层次决策,权重难确定 | 科学严谨 | 计算复杂 |
| TOPSIS | 大量方案,客观排序 | 客观,适合多方案 | 需要量化数据 |
| 决策树 | 顺序决策,有不确定性 | 可视化好 | 概率估计难 |

## 敏感度分析

系统性评估决策稳健性:

### 权重敏感度
- 改变关键标准权重(±10%)
- 观察排序变化
- 识别关键影响因素

### 评分敏感度
- 评分±1分或±2分
- 评估误差容忍度
- 识别关键方案

### 情景分析
- 最佳情景 (乐观)
- 最差情景 (悲观)
- 最可能情景 (现实)

## 与其他技能集成

### 集成模式1: 生成-评估
```
problem-solving → 生成多个解决方案
  ↓
decision-support → 评估和选择最优方案
```

### 集成模式2: 风险调整决策
```
decision-support → 初步评估
  ↓
risk-assessment → 详细风险分析
  ↓
decision-support → 风险调整推荐
```

### 集成模式3: 综合决策
```
decision-support → 识别备选方案
  ↓
并行调用:
├─ risk-assessment (风险评估)
└─ cost-benefit-analysis (经济分析)
  ↓
decision-support → 整合所有分析 → 综合推荐
```

## 决策陷阱预防

本技能帮助避免常见决策陷阱:

- ❌ 锚定偏差 → ✅ 独立评估
- ❌ 确认偏差 → ✅ 主动寻找反对证据
- ❌ 沉没成本 → ✅ 只考虑未来
- ❌ 过度自信 → ✅ 参考数据和专家意见
- ❌ 群体思维 → ✅ 鼓励多元观点
