# Agents Directory

This directory contains specialized agent configurations for Claude Code.

## What are Agents?

Agents are specialized AI assistants configured for specific types of tasks. They have custom instructions, tool access, and behavior patterns optimized for their domain.

## Types of Agents

Agents can be created for various purposes:

- Code review agents
- Documentation agents
- Testing agents
- Refactoring agents
- Security analysis agents
- And more...

## Structure

Each agent configuration should include:

- Agent purpose and capabilities
- Tool access permissions
- Behavioral guidelines
- Usage examples
- Performance considerations

## Creating Agents

When creating a new agent:

1. Define the agent's specific purpose
2. Configure appropriate tool access
3. Set behavioral guidelines
4. Test thoroughly with representative tasks
5. Document usage patterns
